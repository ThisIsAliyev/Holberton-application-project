WELCOME TO MY WEBSITE FOR

All products page

The "All Products" page of this website allows users to browse, filter (not activated), and explore a wide range of products in detail. Structurally, the page is divided into three main sections: Header, Main, and Footer. Header Located at the top of the page, the header provides essential navigation tools. On the left side, you'll find the "Aliyev.co" logo. Clicking on it at any time redirects the user back to the homepage (Welcome Page). Next to the logo are navigation links labeled “Shop”, “On Sale”, “New Arrivals”, and “Brands”. Although these links are currently inactive, they are placeholders for future features and categories. On the right side, there is a search input field for finding products, a cart icon (currently non-functional), and a link to the user profile page. By clicking on the profile section, users are redirected to the sign in/sign up page. Main Section This is the core of the page and is split into two columns: On the left, there is a filter panel that allows users to refine products based on specific criteria. On the right, the products grid displays all available items. Clicking on any product provides more detailed information about it. Responsive Design The layout has been fully optimized for smaller screens (such as mobile phones and tablets) to ensure a smooth user experience: The filter panel is hidden by default and is accessible via an icon located on the right-hand side of the screen. Tapping this icon reveals the filters, allowing users to apply their preferences easily. The navigation links (“Shop”, “On Sale”, “New Arrivals”, and “Brands”) are collapsed into a compact menu icon, and the “Aliyev.co” logo is shifted to the right for a cleaner, more streamlined appearance. Users can access the menu items by clicking on the icon. Footer Positioned at the bottom of the page, the footer contains contact information, social media links, and other helpful resources.

Sign up/in page

The login page is designed to allow users to securely access their accounts. The layout consists of three main sections: Header, Main, and Footer. Header The header appears at the top of the page and maintains a consistent design with other pages on the site. On the left, the "Aliyev.co" logo is displayed. Clicking on it redirects the user back to the homepage (Welcome Page). On the right, there is a link to the profile section. Main Section This central section provides the elements necessary for users to log in: A welcoming heading greets the user. An email input field where users can enter their email address. A password input field for users to enter their password. A "Forgot Password?" link is present but currently inactive — intended for future password recovery functionality. A Login button that submits the form once the credentials are entered. A Sign Up option is also available, but it only functions under the following conditions: Both the email and password fields must be filled. The password must be at least 8 characters long. If the email is missing or entered in an invalid format, an error message is displayed and the sign-up is prevented. Footer Located at the bottom of the page, the footer includes contact details, social media links, and other useful information. Responsive Design The login page is fully optimized for mobile and smaller screen sizes. Input fields and the login button are scaled and spaced appropriately to ensure a smooth and comfortable experience across all devices.

Product Details

The Product Details Page provides users with detailed information about a specific product selected from the "All Products" page. Unlike other pages on the site, this page cannot be accessed directly via a static link, as it dynamically retrieves product data based on the item selected from the product listing (via API). Therefore, users must first navigate to the All Products page, select a product, and only then are they redirected to this detailed view. This page is divided into three main sections: Header, Main, and Footer. Header The header remains consistent with other pages on the site. On the left side, the "Aliyev.co" logo is displayed. Clicking it always redirects the user to the homepage (Welcome Page). Next to the logo are navigational links such as “Shop,” “On Sale,” “New Arrivals,” and “Brands.” These are currently non-functional and serve as placeholders for future expansion. On the right side, there's a search bar, a shopping cart icon (non-functional for now), and a profile link where users can log in or sign up. Main Section This section is designed to showcase all the key details of a selected product. The layout follows a two-column structure: On the left, there is a product image showing a visual preview of the selected item. On the right, the following product details are displayed in vertical order: Product Title at the top Rating section underneath the title Price shown below the rating Product Description providing key features or specs Delivery Time information showing estimated shipping duration At the bottom-right of the product details section, there is an "Add to Cart" button. However, this button is currently non-functional and does not perform any action yet. Footer Located at the bottom of the page, the footer includes contact information, social media links, and additional resources — similar to other pages across the site. Note on Development Due to time constraints, only a basic version of the product page has been implemented so far. The design and functionality remain work-in-progress and can be enhanced significantly in future iterations — including making the "Add to Cart" button functional, refining the layout, and improving responsiveness.

Tweets

The Tweets Page is designed to showcase selected embedded tweets related to the company "Aliyev.co", with a specific focus on project announcements, updates, and other relevant social media interactions. Just like other pages, this one is structured into three primary sections: Header, Main, and Footer. Header: The header retains the consistent layout used throughout the website. On the left side is the "Aliyev.co" logo, which redirects users to the homepage (Welcome Page) when clicked. Next to the logo are navigation links — “Shop”, “On Sale”, “New Arrivals”, and “Brands”. While currently inactive, these links are placeholders for future navigation options. On the right, users will find a search bar, a shopping cart icon (not functional at this stage), and a profile link for login or sign-up purposes. Main Section: This section features a vertically arranged feed of embedded tweets directly from the official X (formerly Twitter) account of the company. By integrating live tweets, the page allows users to follow the latest updates and engage with ongoing developments without leaving the website. Responsive Design: On smaller devices, tweets are resized appropriately to maintain readability and design consistency. Navigation links collapse into a menu icon for better mobile usability. Footer: The footer contains standard information including contact details, social media links, and access to additional resources — matching the layout of other pages.
